## Premiers pas

### Créer un compte

1. Rendez-vous sur [dreamoracle.eu](https://dreamoracle.eu)
2. Cliquez sur **"Commencer maintenant"** ou **"S'inscrire"**
3. Remplissez le formulaire avec :
   - Votre nom
   - Votre adresse email
   - Un mot de passe sécurisé
4. Cliquez sur **"Créer mon compte"**

### Se connecter

1. Cliquez sur **"Se connecter"**
2. Entrez votre email et mot de passe
3. Vous arrivez sur votre tableau de bord personnel

## Enregistrer un rêve

### Accéder au formulaire

Depuis le tableau de bord, cliquez sur **"Nouveau rêve"** ou l'icône **+**.

### Remplir les informations

#### Titre du rêve
Donnez un titre court et évocateur à votre rêve.
> Exemple : "Vol au-dessus de l'océan", "La maison d'enfance"

#### Description du rêve
C'est le champ le plus important. Décrivez votre rêve avec le plus de détails possible :
- Les lieux
- Les personnes présentes
- Les actions
- Les couleurs, sons, sensations
- Ce qui vous a marqué

> **Conseil** : Notez votre rêve dès le réveil, avant que les détails ne s'effacent !

#### Date du rêve
Sélectionnez la date à laquelle vous avez fait ce rêve.

#### Émotions ressenties
Cochez les émotions que vous avez ressenties pendant ou après le rêve :
- Joie, Peur, Tristesse, Colère
- Surprise, Confusion, Paix, Anxiété
- Amour, Nostalgie, Émerveillement, Culpabilité

#### Qualité du sommeil
Notez de 1 à 5 la qualité de votre nuit de sommeil.

#### Niveau de lucidité
Indiquez de 0 à 5 à quel point vous étiez conscient de rêver :
- **0** : Pas du tout conscient
- **5** : Totalement lucide (contrôle du rêve)

#### Humeur au réveil
Comment vous sentiez-vous en vous réveillant ?
- Positif, Négatif, Neutre, Confus

#### Rêve récurrent
Cochez cette case si c'est un rêve que vous faites régulièrement.

#### Tags personnalisés
Ajoutez vos propres mots-clés pour organiser vos rêves.
> Exemples : `voyage`, `famille`, `cauchemar`, `eau`, `vol`

### Sauvegarder

Cliquez sur **"Enregistrer le rêve"** pour sauvegarder.

## Demander une interprétation

### Première interprétation

1. Ouvrez un rêve enregistré
2. Descendez jusqu'à la section **"Demander une interprétation"**
3. Choisissez un **style d'interprétation** :

| Style | Description |
|-------|-------------|
| **Spirituel** | Basé sur la symbolique ésotérique, les archétypes universels et les traditions mystiques |
| **Psychologique** | Analyse selon les principes de Jung : ombre, anima/animus, processus d'individuation |
| **Équilibré** | Combinaison harmonieuse des approches spirituelle et psychologique |

4. Cliquez sur **"Consulter l'Oracle"**
5. Patientez quelques secondes pendant que l'IA analyse votre rêve

### Comprendre l'interprétation

L'Oracle vous fournit :

- **Symboles clés** : Les éléments importants de votre rêve et leur signification
- **Thèmes émotionnels** : Les sentiments sous-jacents identifiés
- **Message principal** : Ce que votre subconscient essaie de vous communiquer
- **Questions de réflexion** : Des pistes pour approfondir votre compréhension
- **Conseil pratique** : Une suggestion concrète à appliquer dans votre vie

### Voir d'autres perspectives

Après avoir reçu une interprétation, vous pouvez la voir sous un autre angle :

1. Repérez les boutons **"Spirituel"**, **"Psychologique"** ou **"Équilibré"** sous l'interprétation
2. Cliquez sur le style souhaité
3. Une nouvelle interprétation sera générée avec cette approche

Cela vous permet de comparer les différentes lectures de votre rêve.

## Modifier un rêve

Si vous avez oublié des détails ou souhaitez enrichir votre description :

1. Ouvrez le rêve concerné
2. **Survolez** le texte de votre rêve avec la souris
3. Un bouton **"Modifier"** apparaît en haut à droite
4. Cliquez dessus pour passer en mode édition
5. Modifiez votre texte
6. Cliquez sur **"Sauvegarder"** ou **"Annuler"**

Après modification, vous pouvez demander une nouvelle interprétation qui prendra en compte les nouveaux détails.

## Gérer vos rêves

### Voir tous vos rêves

Cliquez sur **"Mes rêves"** dans le menu pour accéder à la liste complète de vos rêves enregistrés.

### Consulter un rêve

Cliquez sur n'importe quel rêve de la liste pour voir :
- Le contenu complet
- Les métadonnées (émotions, qualité du sommeil, etc.)
- L'interprétation si elle existe

### Supprimer un rêve

1. Ouvrez le rêve à supprimer
2. Cliquez sur le bouton **corbeille** (en haut à droite)
3. Confirmez la suppression

> **Attention** : Cette action est irréversible !

## Les paramètres

Accédez aux paramètres via l'icône engrenage ou le menu **"Paramètres"**.

### Style d'interprétation par défaut
Choisissez votre style préféré qui sera pré-sélectionné pour chaque nouvelle interprétation.

### Langue
Sélectionnez la langue de l'interface (français par défaut).

### Thème visuel
Personnalisez l'apparence de l'application :
- Dark
- Midnight
- Aurora

## Astuces pour de meilleures interprétations

### 1. Notez vos rêves immédiatement
Les détails s'effacent rapidement. Gardez un carnet ou votre téléphone près du lit.

### 2. Soyez précis dans la description
Plus vous donnez de détails, plus l'interprétation sera pertinente :
- Couleurs, textures, sons
- Dialogues ou pensées
- Sensations physiques
- Transitions entre les scènes

### 3. Incluez vos émotions
Les émotions ressenties dans le rêve sont souvent la clé de l'interprétation.

### 4. Notez le contexte de votre vie
Même sans l'écrire dans l'application, gardez en tête ce qui se passe dans votre vie. Les rêves y font souvent écho.

### 5. Comparez les styles d'interprétation
Chaque perspective (spirituelle, psychologique, équilibrée) peut révéler des aspects différents du même rêve.

### 6. Utilisez les tags
Créez votre propre système de classification pour repérer les thèmes récurrents au fil du temps.

### 7. Relisez vos anciens rêves
Avec le recul, certaines interprétations peuvent prendre un nouveau sens.
